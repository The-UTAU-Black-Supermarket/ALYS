* What is ALYS DB 002 FRA?
ALYS DB 002 FRA is the second French vocal library developped for ALYS
from August 2014 to mid-2015. It was ALYSâ€™ first publicly heard vocal
library. It is developed as an UTAU vocal library.

* Download and Installation
Either download this repository through ~git~ or its archive from [[https://labs.phundrak.com/ALYS/ALYS-DB-002-FRA/releases][the
release page]] and place the directory in your ~voicebanks~ directory in
the UTAU installation directory. You can then chose ALYS from your
vocal library list in UTAU.
